# Проект: Flameflow — лендинг-портфолио

## Что это
Статический одностраничный сайт-визитка (HTML + CSS + чистый JS, без фреймворков и сборщиков). Публикуется через GitHub Pages. Автор — Владислав, фрилансер. Владелец правит код в VS Code на Mac, деплой через git push в main.

## Кто автор и позиционирование
Владислав — специалист по Telegram-ботам для ВНУТРЕННИХ процессов бизнеса (не клиентские боты). Ядро специализации, подтверждённое реальными кейсами:
- Ответы по документам (бот-база знаний: сотрудник спрашивает — бот находит ответ во внутренних документах)
- Контроль задач (постановка задач, напоминания, контроль сроков)
- Разбор обращений (бот разбирает входящий поток, сортирует, отдаёт сводку — снимает ручную рутину с сотрудника)

Позиционирование строится ТОЛЬКО на подтверждённых фактах. Не заявлять услуги и результаты, которых нет. Специализация узкая и конкретная, но не «намертво» — можно шире в рамках ботов.

## Правила текстов (важно)
- Язык бизнеса, а не технический жаргон (клиенты, задачи, документы — да; aiogram, API — нет).
- НЕ использовать тире (—) в текстах. Оно выдаёт ИИ. Перестраивать фразы без него.
- Не писать «ИИ-шно»: без оборотов вроде «в современном мире», «поможет вывести на новый уровень», без пустого пафоса.
- Без оправданий и заискивания («не берусь за всё подряд», «я лично отвечаю за результат» — убирать).
- Прибыль/польза подаётся через механику, а не голословные цифры. Не «+30% продаж», а «не теряете заявки = не теряете деньги».
- Тексты короткие, по делу, живые. Каждое слово должно нести смысл.

## Технические факты
- Тёмная тема. Фон #0d0d1a. Акцент синий #1d4ed8, свечения в синих тонах. Оранжевый НЕ использовать (пробовали, убрали).
- Видео встраивается переключателем YouTube/RuTube (функция switchVideo в конце index.html). RuTube в iframe не всегда играет — это известно, YouTube основной.
- Вертикальные видео (шортсы) — класс .video-container.vertical, горизонтальные — .horizontal.
- Адаптив под мобильные через @media (max-width: 600px) и (max-width: 900px).
- Кнопка «Написать в Telegram» ведёт на https://t.me/flameflow.
